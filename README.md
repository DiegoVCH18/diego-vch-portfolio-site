## 🗂️ Estructura del proyecto

```
diego-vch-portfolio-site/
├── index.html              # Página principal (homepage)
├── projects.html           # Listado de proyectos de consultoría

│   ├── academic-performance-prediction.html
│   ├── automatizacion-reportes-mibanco.html
│   ├── customer-experience-loyalty-dashboard.html
```

## 📁 Proyectos destacados

### 🥇 IAhorra CERTUS — Evolución 2025 a 2.0
`React · Next.js · Vercel · IA Conversacional · EdFinTech`  
**IAhorra 2025:** asistente de IA conversacional ganador del **1.er lugar nacional en Propuesta Innovadora de Educación Financiera** (Semana Mundial del Ahorro 2025, SBS).  
**IAhorra 2.0:** solución EdFinTech completa, actualmente presentada como Propuesta Innovadora de Educación Financiera para la Semana Mundial del Ahorro 2026 (SBS).  
Educación financiera socrática para niños, jóvenes y emprendedores citando fuentes SBS y ASBANC.  
🔗 [iahorra-certus.vercel.app](https://iahorra-certus.vercel.app) · [Código](https://github.com/DiegoVCH18/iahorra-certus)

---

### 🎓 Academic Performance Prediction — CERTUS
`Python · scikit-learn · XGBoost · CatBoost · HuggingFace Spaces`  
Modelo predictivo de rendimiento y permanencia estudiantil sobre ~7,000 registros por ciclo (sistema ARGOS). Predice desde la Evaluación 1 si un estudiante aprobará, reprobará o abandonará con **+90% de precisión**.  
🔗 [HuggingFace Spaces](https://huggingface.co/spaces/Diegovch18/PROYECTO_CERTUS)

---

### 📊 Customer Experience & Loyalty Dashboard
`Power BI · DAX · Data Storytelling`  
Dashboard ejecutivo con CSAT, NPS, Permanence Index y análisis de riesgo de churn. Storytelling orientado a decisiones de servicio al cliente para gerencia.

---

### ⚙️ Automatización de Reportes — MIBANCO
`Excel VBA · Power Query · Apps Script`  
Sistema de reportería masiva con macros VBA para consolidación mensual, reduciendo **más del 60%** el tiempo de reporte. Aprobado por la Gerencia de Aprendizaje de MIBANCO.

---

### 🏦 Simulador Bancario SICA
`Excel · VBA`  
Simulador educativo con lógica bancaria real: depósitos, retiros, transferencias y cheques. Diseñado para el programa de Negocios Bancarios y Financieros (Zegel / InLearning).

## 🚀 Ejecutar en local

```bash
# Instalar dependencias
npm install

# Build de producción (genera /dist)
npm run build
```

El sitio es estático: basta abrir `index.html` directamente en el navegador para desarrollo local, sin necesidad de servidor.
